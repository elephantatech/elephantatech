<div align="center">
  <h3>Moon Traveler Terminal</h3>
  <p><i>A text-based survival game on Saturn's moon Enceladus with LLM-powered alien conversations</i></p>
  <p>
    <a href="https://elephantatech.github.io/moon_traveler/"><img src="https://img.shields.io/badge/Play_Now-c89450?style=for-the-badge" /></a>
    <a href="https://github.com/elephantatech/moon_traveler"><img src="https://img.shields.io/badge/GitHub-181d30?style=for-the-badge&logo=github&logoColor=white" /></a>
    <a href="https://github.com/elephantatech/moon_traveler/releases"><img src="https://img.shields.io/badge/Download-8ca4be?style=for-the-badge" /></a>
  </p>
</div>

<table border="0" width="100%">
<tr>
<td width="50%" valign="top">

**Crash-landed on Enceladus. No signal. No rescue.** Talk your way off an alien moon — or die on the ice.

- Local LLM-powered creature dialogue (Qwen 3.5 / Gemma 4)
- 10 creature archetypes with trust-based relationships
- NPC memory system — creatures remember you across conversations
- Procedural world generation with 3 game modes
- Hostile environment with geyser eruptions, ice storms, and crevasses
- Ship bays, escort system, trade system, drone upgrades
- Cross-platform: macOS, Linux, Windows — runs 100% offline

</td>
<td width="50%" valign="top">

**Quick Install:**

```bash
# macOS / Linux
curl -fsSL https://raw.githubusercontent.com/elephantatech/moon_traveler/main/install.sh | bash

# Windows (PowerShell)
irm https://raw.githubusercontent.com/elephantatech/moon_traveler/main/install.ps1 | iex
```

**From source:**
```bash
git clone https://github.com/elephantatech/moon_traveler.git
cd moon_traveler && uv sync && python play.py
```

</td>
</tr>
</table>
